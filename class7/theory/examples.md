# Приклади до теми «Цифрові пристрої»

## Порівняння цифрових пристроїв

### Смартфон, ноутбук та планшет: що обрати?

| Характеристика | Смартфон | Планшет | Ноутбук |
|---|---|---|---|
| **Портативність** | Завжди в кишені | Легко взяти з собою | Потрібна сумка |
| **Екран** | 5–7 дюймів | 8–13 дюймів | 13–17 дюймів |
| **Клавіатура** | Віртуальна (на екрані) | Віртуальна / окрема | Вбудована |
| **Продуктивність** | Середня | Середня | Висока |
| **Батарея** | 1–2 дні | 8–12 год | 4–8 год |
| **Зв'язок** | SIM, Wi-Fi, Bluetooth | Wi-Fi, Bluetooth, (LTE) | Wi-Fi, Bluetooth |
| **Зручність для тексту** | Низька | Середня | Висока |
| **Зручність для відео** | Середня | Висока | Висока |
| **Зручність для ігор** | Казуальні | Середня | Висока |
| **Ціна** | 3000–40000 грн | 4000–30000 грн | 8000–60000 грн |

**Приклад задачі:** Учневі потрібно писати реферат, робити презентацію та спілкуватися з однокласниками. Який пристрій найкращий?

**Відповідь:** Ноутбук — зручно набирати текст на клавіатурі, великий екран для презентацій, достатня продуктивність. Смартфон можна використовувати додатково для спілкування.

**Приклад задачі:** Учневі потрібно читати підручник, дивитися відеоуроки та робити нотатки на полях. Який пристрій найкращий?

**Відповідь:** Планшет зі стилусом — зручно читати на великому екрані, стилусом робити нотатки, легко тримати в руках.

## Класифікація програмного забезпечення

### Схема класифікації

```
Програмне забезпечення
├── Системне
│   ├── Операційна система (Windows, macOS, Linux, Android, iOS)
│   ├── Драйвери (драйвер принтера, драйвер відеокарти)
│   └── Утиліти (антивірус, архіватор, дефрагментатор)
├── Прикладне
│   ├── Офісне (Word, Excel, PowerPoint, LibreOffice)
│   ├── Браузери (Chrome, Firefox, Safari, Edge)
│   ├── Комунікаційне (Telegram, Zoom, Discord)
│   ├── Медіа (VLC, Spotify, Photoshop, Audacity)
│   ├── Ігри (Minecraft, Roblox, Chess)
│   └── Освітнє (Kahoot!, Quizlet, Duolingo)
└── Інструментальне
    ├── Мови програмування (Python, JavaScript, C++)
    ├── Середовища розробки (VS Code, PyCharm, Scratch)
    └── Компілятори та інтерпретатори
```

## Опис алгоритму усунення несправностей

### Блок-схема усунення проблеми

```
Початок: комп'ютер працює ненормально
    │
    ▼

    ▼ Ні
Крок 2: Визначте тип проблеми
    │
    ├─── Апаратна? ────────────────────────┐
    │                                      │
    │   Крок 3а:                           Крок 3б:
    │   • Перевірте кабелі та              • Закрийте завислу
    │     підключення                      програму (Диспетчер
    │   • Перевірте живлення               задач)
    │   • Спробуйте інший USB-порт         • Оновіть або
    │   • Перевірте температуру            переінсталюйте
    │                                      програму
    │                                      • Перевірте антивірусом
    │                                      │
    │   Крок 4а:                           Крок 4б:
    │   Проблема зникла?                   Проблема зникла?
    │     │                                   │
    │     ├── Так ──▶ Кінець               ├── Так ──▶ Кінець
    │     ▼ Ні                             ▼ Ні
    │   Крок 5а:                           Крок 5б:
    │   Зверніться до                      Скористайтеся
    │   фахівця                            точкою відновлення
    │                                      │
    │                                      ├── Проблема зникла?
    │                                      │   ├── Так ──▶ Кінець
    │                                      │   └── Ні ──▶ Зверніться
    │                                      │              до фахівця
```

### Приклади типових проблем і рішень

| Проблема | Тип | Рішення |
|---|---|---|
| Принтер не друкує | Апаратна / Програмна | Перевірте кабель, чорнило, чергу друку, драйвер |
| Комп'ютер зависає | Програмна | Закрийте програму через Диспетчер задач, перезавантажте |
| Відсутній звук | Програмна | Перевірте гучність, драйвер, пристрій виведення |
| Wi-Fi не працює | Апаратна / Програмна | Перевірте роутер, пароль, драйвер Wi-Fi адаптера |
| Флешка не відкривається | Апаратна | Спробуйте інший USB-порт, перевірте на іншому комп'ютері |
| Синій екран | Апаратна / Програмна | Запам'ятайте код помилки, скористайтеся точкою відновлення |
| Клавіатура не працює | Апаратна | Перевірте підключення, спробуйте іншу клавіатуру |
| Браузер працює повільно | Програмна | Очистіть кеш, вимкніть зайві розширення |

## Мережеве обладнання: приклади з життя

### Домашня мережа

Уявіть типову домашню мережу:

```
[Провайдер інтернету]
        │
        ▼
    [Модем] ← перетворює сигнал провайдера
        │
        ▼
  [Маршрутизатор (роутер)] ← роздає інтернет по дому
     │    │    │
     ▼    ▼    ▼
  [ПК] [Ноутбук] [Смартфон]
            │
            ▼
        [Комутатор (світч)] ← якщо потрібно більше портів
         │     │
         ▼     ▼
      [Принтер] [Інший ПК]
```

### Мережа в комп'ютерному класі

У шкільному комп'ютерному класі мережа побудована інакше:

```
[Сервер школи]
      │
      ▼
  [Маршрутизатор] ← керує трафіком між класами
      │
      ▼
  [Комутатор (світч)] ← з'єднує всі комп'ютери класу
   │  │  │  │  │
   ▼  ▼  ▼  ▼  ▼
  [ПК1] [ПК2] ... [ПК12] ← комп'ютери учнів
```

### Порівняння мережевих пристроїв

| Пристрій | Кількість підключень | Функція | Де використовується |
|---|---|---|---|
| Роутер | 4–8 портів + Wi-Fi | Поєднує мережі, роздає інтернет | Будинок, малий офіс |
| Світч | 5–48 портів | Поєднує комп'ютери в одній мережі | Школа, офіс |
| Модем | 1–2 порти | Перетворює сигнал провайдера | Будинок, квартира |

## Вибір комп'ютерної системи: практичні задачі

### Задача 1: Комп'ютер для навчання

**Умова:** Учень 7 класу потребує комп'ютер для виконання домашніх завдань, пошуку інформації в інтернеті та створення презентацій. Бюджет — 12 000 грн.

**Рішення:**

| Компонент | Рекомендація | Чому |
|---|---|---|
| Процесор | Intel Core i3 / AMD Ryzen 3 | Достатньо для навчальних задач |
| RAM | 8 ГБ | Комфортно для браузера та офісних програм |
| Накопичувач | SSD 256 ГБ | Швидкий запуск програм |
| Відеокарта | Вбудована | Не потрібна для навчання |
| Монітор | 21–24 дюйми | Зручно для презентацій |
| Приблизна вартість | ~10 000–12 000 грн | Вписується в бюджет |

### Задача 2: Комп'ютер для відеомонтажу

**Умова:** Старшокласник займається створенням відео для YouTube. Який комп'ютер йому потрібен?

**Рішення:**

| Компонент | Рекомендація | Чому |
|---|---|---|
| Процесор | Intel Core i7 / AMD Ryzen 7 | Відеомонтаж потребує потужного процесора |
| RAM | 16–32 ГБ | Відеоредактори використовують багато пам'яті |
| Накопичувач | SSD 512 ГБ + HDD 1 ТБ | SSD для програм, HDD для відеофайлів |
| Відеокарта | Виділена (NVIDIA RTX) | Прискорює рендеринг відео |
| Монітор | 27 дюймів, IPS | Точні кольори для монтажу |

### Задача 3: Апаратний чи програмний підхід?

**Умова:** Комп'ютер став працювати повільно під час роботи браузера з багатьма вкладками. Що робити?

| Підхід | Дія | Вартість | Результат |
|---|---|---|---|
| Апаратний | Додати 8 ГБ RAM (було 4 ГБ) | ~500–800 грн | Браузер працюватиме швидко |
| Програмний | Встановити легший браузер (Firefox), закривати непотрібні вкладки | Безкоштовно | Помітне прискорення, але обмежене |
| Комбінований | Додати RAM + оптимізувати роботу браузера | ~500–800 грн | Найкращий результат |

**Висновок:** Спочатку спробуйте програмний підхід (безкоштовно). Якщо не допоможе — тоді апаратний.
