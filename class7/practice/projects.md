# Проекти

## Проект 1. Мій цифровий світ

**Мета:** Дослідити цифрові пристрої у своєму домі, класифікувати їх за призначенням та типом.

### Завдання

1. Обійдіть свій дім і запишіть усі цифрові пристрої, які ви знайдете.
2. Для кожного пристрою визначте:
   - Назву (наприклад, смартфон, ноутбук, розумний годинник)
   - Тип за призначенням (для комунікації, для навчання, для розваги, для роботи)
   - Тип сигналів (цифровий або аналоговий)
   - Які типи даних опрацьовує (текст, зображення, звук, відео)
3. Створіть таблицю класифікації знайдених пристроїв.
4. Визначте, який пристрій ви використовуєте найчастіше, і поясніть чому.

### Очікуваний результат

Таблиця з переліком не менше 8 цифрових пристроїв, класифікованих за типом та призначенням. Короткий висновок (3-4 речення) про те, які пристрої найважливіші у вашому житті.

| Назва пристрою | Призначення | Тип сигналів | Типи даних |
|---|---|---|---|
| Смартфон | Комунікація, розвага | Цифровий | Текст, зображення, звук, відео |
| ... | ... | ... | ... |

## Проект 2. Безпечний комп'ютер

**Мета:** Створити контрольний перелік правил безпеки та ергономіки для свого робочого місця за комп'ютером.

### Завдання

1. Осмотріть своє робоче місце за комп'ютером.
2. Перевірте виконання правил ергономіки:
   - Екран на рівні очей чи нижче?
   - Відстань до екрану 50-70 см?
   - Спина пряма, плечі розслаблені?
   - Лікті під кутом 90 градусів?
   - Освітлення достатнє, без відблисків на екрані?
3. Перевірте налаштування безпеки на вашому комп'ютері чи смартфоні:
   - Чи встановлено надійний пароль?
   - Чи увімкнено двофакторну автентифікацію?
   - Чи встановлено антивірус?
   - Чи оновлено операційну систему?
   - Чи є резервні копії важливих даних?
4. Створіть контрольний перелік (чек-лист) із двох частин: «Ергономіка» та «Безпека».
5. Відмітте пункти, які ви вже виконуєте, і пункти, які потрібно покращити.

### Очікуваний результат

Заповнений контрольний перелік із позначками «виконано» та «потрібно покращити». План із 3-5 конкретних кроків для покращення безпеки та ергономіки вашого робочого місця.

**Приклад контрольного переліку:**

**Ергономіка:**
- [ ] Екран на рівні очей
- [ ] Відстань до екрану 50-70 см
- [ ] Спина пряма, ноги на підлозі
- [ ] Лікті під кутом 90 градусів
- [ ] Добре освітлення без відблисків
- [ ] Перерва кожні 45-60 хвилин

**Безпека:**
- [ ] Надійний пароль (8+ символів)
- [ ] Двофакторна автентифікація
- [ ] Антивірус встановлено та оновлено
- [ ] Операційна система оновлена
- [ ] Резервні копії даних
- [ ] Кеш очищено

## Проект 3. Порівняння пристроїв

**Мета:** Порівняти два цифрові пристрої за основними характеристиками та визначити, який краще підходить для конкретних завдань.

### Завдання

1. Оберіть два пристрої для порівняння (наприклад: смартфон vs планшет, ноутбук vs настільний комп'ютер, розумний годинник vs фітнес-браслет).
2. Для кожного пристрою визначте такі властивості:
   - Портативність (чи легко взяти з собою)
   - Продуктивність (наскільки потужний)
   - Екран (розмір, якість)
   - Автономність (час роботи від батареї)
   - Функціональність (які задачі виконує)
   - Вартість (приблизна цінова категорія)
3. Заповніть таблицю порівняння.
4. Для кожного з трьох завдань визначте, який пристрій краще підходить:
   - Написання реферату на 5 сторінок
   - Перегляд фільму в дорозі
   - Спілкування з друзями в месенджері
5. Напишіть висновок: для яких завдань краще перший пристрій, а для яких — другий.

### Очікуваний результат

Таблиця порівняння двох пристроїв за вказаними властивостями. Обґрунтований вибір пристрою для трьох конкретних завдань. Висновок (4-5 речень) про те, що не існує «найкращого» пристрою — є найкращий для конкретної задачі.

**Приклад таблиці порівняння:**

| Властивість | Смартфон | Планшет |
|---|---|---|
| Портативність | Дуже висока | Середня |
| Продуктивність | Середня | Середня |
| Екран | Маленький (5-6") | Великий (10-12") |
| Автономність | 1-2 дні | 8-10 годин |
| Клавіатура | Віртуальна | Віртуальна / зовнішня |
| Функціональність | Дзвінки, повідомлення, камера, навігація | Читання, відео, ігри, навчання |
