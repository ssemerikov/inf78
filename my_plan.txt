Лабораторна робота № 2
Розробка навчальних матеріалів з інформатики з використанням генеративного штучного інтелекту
Мета

Створити повноцінний навчальний контент з обраної теми інформатики для учнів 10-11 класів з використанням можливостей генеративного штучного інтелекту, розмістити результат у репозиторії GitHub з публікацією на GitHub Pages та надати на них посилання у Moodle. 
Завдання
1. Визначення теми та планування

    Обрати тему з інформатики для 10-11 класів (з наданого переліку або запропонувати власну)
    Визначити, чи тема відноситься до базової програми або призначена для поглибленого вивчення
    Сформулювати мету уроку/теми та очікувані результати навчання
    Розробити детальний план теми (підтеми, ключові поняття, практичні завдання)

2. Генерація навчального контенту з використанням ШІ

2.1. Текстовий контент:

    Використати великі мовні моделі (GPT-4, Claude, Gemini тощо) для створення:
        Теоретичного матеріалу з поясненнями та прикладами
        Інструкцій до практичних завдань
        Глосарію термінів
        Додаткових матеріалів для самостійного вивчення

2.2. Візуальний контент:

    Застосувати генеративні ШІ (Midjourney, DALL-E, Stable Diffusion тощо) для створення:
        Ілюстрацій та схем
        Інфографіки
        Діаграм та блок-схем
        Візуальних прикладів

2.3. Інтерактивні елементи:

    Розробити з допомогою ШІ:
        Тести для самоперевірки (різні типи питань)
        Практичні завдання різного рівня складності
        Кейси та ситуаційні задачі
        Творчі завдання та проекти

3. Технічна реалізація на GitHub

3.1. Створення репозиторію:

    Створити публічний репозиторій на GitHub
    Налаштувати структуру папок (теорія, практика, ресурси, тести тощо)
    Написати детальний README.md з описом проекту

3.2. Оформлення матеріалів:

    Конвертувати всі матеріали у формат Markdown
    Додати HTML/CSS для покращення відображення (за потреби)
    Оптимізувати зображення та мультимедіа файли
    Забезпечити коректне відображення формул та коду

3.3. Налаштування GitHub Pages:

    Активувати GitHub Pages для репозиторію
    Створити зручну навігацію по сайту
    Забезпечити адаптивність для різних пристроїв
    Протестувати функціонування всіх посилань та елементів

5. Презентація результатів
